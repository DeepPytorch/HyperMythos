# HyperMythos
This is one of, if not the most powerful AI model ever created. This model may, if fully trained, be near AGI, more powerful than Claude Mythos, and could significantly exceed human capabilities in a large number of fields.
However, this model is useless to the general public, because almost nobody can train it, because most people are not billionares.
# The Alignment Problem
This model could only be trained on a <u>specific</u> type of data, that may possibly fix the problem of alignment.
When a model of this size (2 Trillion parameters) is trained, it picks up on every single piece of bias in the data. The only way a model of this size could be trained, because of its immense power and size, while still keeping it aligned with human values is to both use a large neural network to monitor its thinking, and by engineering a bias in the data, to keep the model away from ideas such as escaping containment, and to keep it away from telling people illegal stuff such as how to make weapons.
